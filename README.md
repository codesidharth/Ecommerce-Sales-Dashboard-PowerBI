# 📊 E-Commerce Sales Dashboard (Power BI)

## 🚀 Project Overview

This project presents an interactive Power BI dashboard built to analyze e-commerce sales performance. It provides insights into revenue trends, profitability, customer behavior, and operational efficiency.

---

## 📌 Key Features

* 📈 **Sales Trend Analysis** (Month-over-Month & Year-over-Year)
* 💰 **KPI Metrics**

  * Total Sales
  * Profit Margin %
  * Sales YoY %
  * Return Rate %
* 🌍 **Regional Performance Analysis**
* 🏆 **Top Products by Sales**
* 🔁 **Returns Analysis**
* 💳 **Payment Method Insights**
* 🎛️ **Interactive Filters (Slicers)**

---

## 🧠 Data Modeling

* Implemented a **Star Schema**
* Fact Table: `ecommerce_sales`
* Dimension Table: `DateTable`
* Created relationships for time intelligence

---

## 📊 DAX Measures Used

### 🔹 Base Metrics

* Total Sales
* Total Orders
* Total Quantity
* Avg Order Value

### 🔹 Profitability

* Total Profit
* Profit Margin %

### 🔹 Returns

* Returned Orders
* Return Rate %

### 🔹 Time Intelligence

* Sales LY (Last Year)
* Sales YTD
* Sales PM (Previous Month)
* Sales YoY %
* Sales MoM %

### 🔹 Filtered Metrics

* Non Returned Sales
* Discounted Sales
